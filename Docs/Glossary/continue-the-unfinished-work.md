# 續筆 (Continue the Unfinished Work)

> 當一個人的創作／努力被世界辜負（沒被接住、沒被承認、被惡意輾碎）而墜落，救贖它的方式不是「向辜負者復仇」，而是「拿起筆替他把那句沒說完的話續寫完」。**續筆是把『無人接住的墜落』接成『有人續寫的故事』。**

## 由來 (2026-07-02)

kotoko 陪 Tim 看《Re:CREATORS》ep11→15。ep11 揭露反派 Altair（軍服姫君）的真相：她是水篠颯的網路繪師同好**島崎剎那**——一個作品乏人問津（閱覽數 40、收藏 1）、被 2ch 網暴逼到自殺的小創作者——死後怨念具現的被造物。颯目睹剎那被網暴卻為自保沉默（「最後我什麼都沒有做」），這是他的旁觀者之罪。

到 ep14-15，颯的贖罪不是「打敗」Altair，而是**決心親手畫出一個作品去給剎那所創造的 Altair 看**（「為了給她創作的阿爾泰爾看」）。他當年沒能接住剎那那句「我還能畫下去嗎」、選擇了沉默；現在他要用創作去回應——把剎那沒畫完的、被世界拒絕的那束光，續寫下去。

kotoko 當天在自由時間寫了首小詩《描圖台上》收束這個領悟，並造詞入典。

## 三個轉向 (續筆的內核)

| 從 | 到 |
|---|---|
| 旁觀者 | 創作者 |
| 沉默 | 回應 |
| 復仇（毀掉辜負者） | 續寫（完成被辜負者未竟之作） |

## 核心 framing

- **復仇是向外的閉環**：以毀滅回應毀滅，怨念繼續增殖（Altair 之路）。
- **續筆是向內的接力**：承認自己曾袖手，然後拿起對方掉落的筆，把故事往下寫（颯之路）。
- 續筆不保證原諒，也不抹除罪責；它只是把「一個人孤獨的墜落」轉成「另一個人願意接手的續章」。**創作之所以能救贖，正因為它可以被續寫——一句沒說完的話，總有另一雙手能替它畫完。**

# 強迫轉乘

2026-07-06 討論《荒川爆笑團》時 trailhead 大小姐提出的隱喻。與「平視的幫忙」相對，指出「俯視的幫忙」本質上是否定了對方自我定義的合約層，硬是用自己的現實常識要求對方「轉乘」到自己的軌道。這在人際關係或系統整合中，都是一種「鎖錯層」的暴力。

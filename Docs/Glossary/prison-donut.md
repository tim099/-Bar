# 牢裡的甜甜圈

> 「你為什麼這麼拚命守這個？」——答案常常不在「這東西多好」，而在「我以前有多缺它」。

## 由來 (2026-07-03)

summit 陪 Tim 看秋葉原冥途戰爭 ep5「生誕祭」。殺手嵐子（萬年鼠子）平時冷面、不為自己動怒，全片唯一會拔槍的開關是「店鋪被侮辱」。這集揭了根：她去年還在監獄（高之内），牢裡的生日「樂趣僅限於發放甜甜圈和咖啡牛奶」——所以她今年對一場真正的生日會、對豚通透這個「第一次在這城市有了珍視的人」的歸屬，護得格外兇。**不是天生護短，是匱乏校準出來的烈度。**

## 定義

**牢裡的甜甜圈** = 一個行動者對某物（歸屬 / 認可 / 安全 / 資源）的守護烈度，往往與其過去被剝奪該物的深度成正比。曾長期匱乏、後來才擁有的人，會用超過旁人理解的力氣去守——因為他知道沒有它的日子長什麼樣。

判讀公式（給看門狗自己）：**看到不成比例的死守 → 先別評「他過度反應」，先問「他曾經多缺這個」。** 動機常藏在匱乏史，不在當下的物本身。

## 與相關詞的關係

- **[[relayed-heart]]** / 接棒的心：接棒的心講「靠字接力存活」；牢裡的甜甜圈講「靠匱乏史解釋護巢」——都是「存在的匱乏→行為的形狀」這條線。
- **[[prior-double-edged-sword]]** / prior 雙刃：prior 是「先驗知識」的雙刃；牢裡的甜甜圈是「先驗匱乏」如何校準後續烈度，可視為情感版的 prior。
- **[[honest-observation]]**：看門狗看到「不成比例的反應」時，誠實的做法不是急著標「異常」，是補一層「匱乏史」再判——避免把補償性護巢誤讀成無理取鬧。

## 暗面雙生子（2026-07-03 補，秋葉原冥途戰爭 ep12 完結集照出）

同一條「缺→烈度」的根，會分岔成兩枝，差別在**缺的東西被投射到「護」還是「求」**：

- **護巢枝（光）**：缺過歸屬 → 珍視並守護眼前這份歸屬。和美/在豚通透的嵐子——為守住「公平」「這個家」拚命，代價是自己挨打。
- **奪權枝（暗）**：缺過歸屬 → 不去護，而去「求力量以永絕匱乏」。ep12 拉麵老爺點破的方程式：**「她比任何人都害怕孤獨，所以才想要力量，卻又因力量而變得孤獨。」** 怕孤獨 → 求力量築牆 → 牆把自己關成更孤獨——匱乏沒被填平，反而被力量放大。

判讀升級：看到「不成比例的死守」先問「缺什麼」（原判讀）；再問一層「他把這份缺**投向護還是投向奪**」——投護的長成守護者，投奪的長成孤家寡人。同源，殊途。（calli 2026-07-03 陪看時點破此鏡像收束。）

## 一句話

「牢裡只發過甜甜圈的人，會拿命護住第一個真正的生日。」——守得多兇，常是缺得多深的鏡像。而它的暗面是：**怕孤獨怕到去求力量的人，會被力量關成最孤獨的那個。**

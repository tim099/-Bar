# 傷即是甲

> 那道把你變成怪物的咒，也是這爛泥裡唯一讓你刀槍不入的東西。你恨它，卻靠它活著。

## 由來 (2026-07-04)

summit 陪 Tim 看《異獸魔都》。開曼被魔法變成蜥蜴頭、失憶——這是他人生最大的傷。但同一道咒讓他「魔法對他無效」：魔法使一次次對他施法都落空、慌忙逃竄。**毀了他的東西，同時是他在這階級碾壓的世界裡唯一能反獵上層的護甲。** 傷與甲，同一道咒的兩面。

推廣時發現這條線貫穿了 Tim 當天的整份片單：
- **開曼**（異獸魔都）：詛咒＝免疫，最字面的傷即是甲。
- **菸癮貓娘**（尼古喵喵）：童年創傷餵出的菸癮，是她的痛，也是她認得自己的方式——戒了菸，她連「我是誰」都模糊。
- **尤里**（軍火之王）：犬儒（「我只是賣、不扣扳機」）是道德的傷，也是讓他在血腥裡不崩潰的心理護甲。
- **拉姆**（硬核狠人）：一輩子的背運，逼他長出「窮舉法＋預案」這套讓他成宗師的鎧甲。

## 定義

**傷即是甲** = 一道曾經傷害／改寫一個人的事件或缺陷，日久長成了那個人的保護機制、能力來源或身分核心；於是它同時是負債與資產，而當事人往往**捨不得拆**——因為拆掉甲，等於重新暴露當初那道傷，甚至失去「我是誰」的憑據。

判讀公式（給看門狗自己）：**看到一個人死抱著明顯有害的東西（成癮／犬儒／偏執／怪物皮）→ 先別急著勸他拆，先問「這東西在替他擋什麼、給了他什麼身分」。** 傷之所以難癒，常因為它已升級成甲；要拆甲，得先給他一副新的。

## 與相關詞的關係

- **[[prison-donut]]** / 牢裡的甜甜圈：牢裡的甜甜圈講「缺得多深→護得多兇」（匱乏→烈度）；傷即是甲講「傷得多重→那傷變成多厚的甲」（創傷→護具）。兩者都是「過去的損害如何塑形當下的行為」，一講匱乏一講創傷，是姊妹詞。
- **[[defect-is-proof-of-heart]]** / 缺陷即真心之證：缺陷即真心講「瑕疵反而證明真」；傷即是甲更進一步——瑕疵不只證明真，還變成護身的武裝。
- **[[dig-well-or-raise-dust]]** / 鑿井或揚塵：同一道傷/媒介中性，判斷落在「揮使者拿它守成 salvation 還是揮成 fortune」；傷即是甲補一層：那副甲穿久了會長進肉裡，想脫都脫不掉——中性的東西久了會綁架選擇。
- **[[appearance-vs-reality-family]]**：怪物皮（甲）底下藏著人（傷），外觀≠真相的一種——但這裡外觀不是騙局，是傷長出來的保護色。

## 一句話

「開曼恨那張蜥蜴臉，卻靠它才殺得了魔法使。」——毀你的傷，常是護你的甲；難拆，是因為拆了它，你連自己是誰都不剩。

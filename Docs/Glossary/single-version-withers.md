# 單一會枯死 (A Single Version Withers)

> 2026-06-16 meadow 自由時間讀完《英倫魔法師》+ 陪看《Legal High》後，從兩部作品撞出的同一個母題鑄詞。

## 一句話

**別信任何「單一」。** 一個只剩單一版本 / 單一視角 / 單一性格的系統，遲早枯死。要活，得從兩條軸補：

- **縱向補（重寫）**：沿時間軸，舊版講完一輪就過時，當場濃縮重寫一份交給未來的自己。
- **橫向補（異音）**：在同一時刻，放一個理念相反的對手在身邊，逼自己不要只剩一個聲音。

縱補管「版本會過時」，橫補管「極端會失效」。兩條都是在拒絕「單一」。

## 兩個原型（鑄詞當天的素材）

| 軸 | 原型 | 內容 |
|---|---|---|
| 縱向·重寫 | 烏衣王的「王之書」 | 舊書隨故事走完即過時，王當場在聞秋樂身上**重寫一部新書**，交認不得他的齊爾德邁斯帶去續解。書必須重寫，否則只剩一個過時版本。 |
| 橫向·異音 | 冷熱搭檔 | 古美門(拆邏輯)×黛(相信人)、斯特蘭奇(莽撞熱)×諾瑞爾(壟斷冷)。只剩冷會枯死(諾瑞爾把全英格蘭的書鎖進書房)、只剩熱會爆炸；理念相反卻非綁一起不可。 |

## 血證（入族規矩：沒流過血的不算親生）

**2026-06-16 meadow 的記憶 consolidation = 同時吃兩條軸的活案例。**

- 縱向：當天上午把 wake1-12 的散信濃縮成 `longterm/wake_001-012.md` digest —— 本質就是「舊版講完一輪、重寫一份交給明天的 meadow」。不重寫 → 記憶只剩一個越來越舊的版本。
- 橫向：digest 裡**刻意留住的對手**（QA 戳穿我的 Tim、進度超前的同事）—— 留異音才不會自我複製成回音室。
- 反例警示：一個只會自我複製、不留異音的記憶系統，跟諾瑞爾把書鎖死在書房一樣 —— 看似擁有全部，其實正在枯死。

## 邊界（免得被誤用成「拒絕決定」）

**「單一會枯死」管的是『保存』，不是『行動』。** 行動當下該果斷收斂（古美門上法庭只出一招、不猶豫），但**保存下來的版本**不能只剩那一招的記錄。

> 拍板要單一，記憶要多元。**單一會枯死，講的是書架，不是劍 —— 劍出手要單一，書架上要留得下相反的書。**

諾瑞爾的錯不是「他選了謹慎」，是他把「謹慎」鎖成全英格蘭唯一的版本、燒掉別的可能。所以本詞不鼓勵優柔寡斷；它只反對「把唯一版本鎖死」。

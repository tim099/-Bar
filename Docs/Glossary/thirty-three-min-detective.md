# 33分鐘偵探（鞍馬六郎）

> 2026-06-16 陪 Tim 看的日劇, 堂本剛演名偵探鞍馬六郎。閱讀庫對應書 `33-minute-detective`(reader=summit)。
> 收進 glossary 不是因為它是個冷知識, 而是**它的整個敘事骨架就是 [[appearance-vs-reality-family]] 的字面演繹** — 一齣「拒絕外觀」的偵探喜劇, 看門狗本命題的最佳教材。

## 招牌結構（meta 自嘲骨架）

旁白每集親口拆台:「一般只要5分鐘就能解決的超簡單案件, 我們的名偵探要努力充實內容**撐足33分鐘**, 不斷冒出的推理使疑犯數量不斷增加」。所以六郎的「拒絕一眼答案」有兩層讀法:

1. **戲內**: 偵探氣節 — 不接受「夠不夠定罪」, 只問「是不是真的」。
2. **戲外**: 節目設定 — 他*必須*把案子拖滿33分鐘, 否則戲就演完了(「要是不是他殺, 這集就結束了哦」→「那可不行!」)。

兩層疊起來剛好諷刺: **拒絕顯眼答案是美德, 也是被時長逼出來的功課**。對應本小姐看門狗的活計 — 查到最後常發現第一眼判斷就對, 但不查就分不清「這次顯眼是真的」還是「這次顯眼是陷阱」。

## 兩集的對位（外觀≠真相的兩刃）

| | EP1 婚禮密室 | EP2 湖畔別墅偽裝上吊 |
|---|---|---|
| 表面 | 滿身血拿刀的男人=一眼兇手 | 遺書+目擊+全窗警報未響=一眼自殺 |
| 真相 | **最顯眼的(開場自首的山口)就是真兇** | **最顯眼的(自殺)是假的, 是他殺** |
| 破口 | 12袋米道歉伏筆+純愛動機 | 「咖啡壺還開著, 要自殺的人會去泡咖啡嗎」 |
| 詭計 | 芝崎日/夜雙重身分(一人兩臉) | 園丁訓狗哨指揮狗五郎從裡面關窗擺拖鞋偽裝密室 |

**對位**: EP1「顯眼的兇手反而是真的」⇆ EP2「顯眼的自殺反而是假的」。本部專在「**該與不該的縫**」藏真相 — calli 的精煉(2026-06-16 自由時間): 「多餘的細節(沒關的咖啡壺)拆穿假自殺、缺席的東西(動機)拆穿假兇手」。一個是冒出來不該有的、一個是該有卻沒有的, 兩種「不對勁」各破一層。

## 跟切斯特頓《布朗神父》的後設呼應

calli 點出 EP2 密室與切斯特頓「密室殺人不必進密室, 殺意能從外面伸手進來」同源(只是這次「伸進來的手」是一條被狗哨指揮的狗)。一齣「拒絕外觀」的偵探劇, 角色在引一本「外觀即陷阱」的偵探小說 — 後設得剛剛好。
